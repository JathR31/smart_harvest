SMS/OTP Setup Guide

Configure Semaphore SMS for farmer verification

Status: SMS service is implemented and ready to use. You just need to add your Semaphore API key to activate it.

1

Get Semaphore API Key

Sign Up

Visit Semaphore website and create an account:

Go to Semaphore.co

Get API Key

  1. Login to your Semaphore dashboard
  2. Navigate to API section
  3. Copy your API Key (starts with letters/numbers)
  4. Keep it secure - don't share publicly

💰 Pricing Info

  • • Free Trial: 100 SMS credits (Free)
  • • Cost per SMS: ₱0.50 - ₱1.00
  • • For 1000 farmers: ~₱750-₱1,500
2

Configure SmartHarvest

Update .env File

Open your .env file and add:

# Semaphore SMS API Configuration
SEMAPHORE_API_KEY=your_api_key_here

Replace your_api_key_here with your actual API key from Semaphore

📍 File Location

c:\xampp\htdocs\dashboard\smart_harvest\.env

3

Test SMS Service

Run Test Script

Open PowerShell/Command Prompt and run:

cd c:\xampp\htdocs\dashboard\smart_harvest
php test_sms.php

How SMS Verification Works

1

Farmer Registers

Farmer fills registration form and selects "SMS Verification"

2

System Generates OTP

6-digit code generated, expires in 10 minutes

3

SMS Sent via Semaphore

OTP code sent to farmer's mobile number

4

Farmer Enters OTP

Farmer types the 6-digit code on verification page

5

Account Verified

Farmer sets password and can access dashboard

Security Features

10-Minute Expiration

OTP codes expire after 10 minutes for security

Rate Limiting

Max 3 resend attempts per minute

Attempt Tracking

Max 5 verification attempts per OTP

Email Fallback

Automatically switches to email if SMS fails

Troubleshooting

❌ "SMS service not configured"

Add SEMAPHORE_API_KEY to your .env file

❌ "Invalid phone number format"

Use Philippine mobile format: 09XXXXXXXXX (11 digits starting with 09)

❌ OTP not received

Check:

  • Phone number is correct
  • Semaphore credits available
  • Network signal on phone
  • Check spam/blocked messages
❌ "Failed to send SMS"

Possible causes:

  • Invalid API key
  • Insufficient credits
  • Network/API error

Check logs: storage/logs/laravel.log
